# Land Registry Демо

Устанавливаем следующие зависимости:

```
npm install --global grunt-cli bower yo generator-karma generator-angular
```

Далее, требуется установить все npm зависимости:

```
npm install
```

И bower зависимости:

```
bower install
```

Запускаем локально:

```
grunt serve
```

Билдим для продакшена:

```
grunt build
```
