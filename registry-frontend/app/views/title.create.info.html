Create a new title

To create a new land title, follow the next steps:

  1. Start planning a title by placing the first point on the map. Note that adding a point within existing land title is not allowed
  2. Use your mouse to make points on the map. Note that making points in a way the line will cross the existing line is not allowed
  3. Finish your land title by making the last poing in your start position
  4. Confirm or decline the created title
